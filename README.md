# BCS-Reverse-N-Money-System
BCS — 双向货币系统 BCS 是一个早期技术原型（Python 实现）。它不挖矿、不投机、不发空气币。 目的 35 岁后就业困难、需求萎缩、经济危机反复——问题不只在人，而在货币只记录“需求”（D），不记录“被需要”（N）。 BCS 在交易规则中增加反向流动： 销售：商家获得 D，同时向买家支付 N 发薪：工人获得 D，同时向雇主支付 N 结果：企业想扩大销售，就必须拥有 N；要获得 N，就必须提供就业。消费与就业通过货币规则重新连接。 欢迎加入 需要开发者（Python/区块链/前端）、研究者（货币/治理/模拟）、试点伙伴（商户/组织）。不承诺暴富，只承诺一个值得认真做的实验。 项目开源，Issue / PR / 邮件均可。
